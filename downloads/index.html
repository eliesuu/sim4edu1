Downloads

You can download the OESjs simulation framework together with a set of simulation examples such that you can modify the examples or create your own web-based simulations.

Download the OESjs ZIP archive and extract its content (for instance, into your "C:" or "D:" partiton, or into a suitable folder like "D:\simulations"). This will create a folder called "OESjs". Then you can

  • run a simulation by visiting its folder and opening its simulation.html file in your web browser.
  • inspect, and possibly modify, the code of a simulation example by opening its simulation.js file with a text editor (like Notepad++) or with a JavaScript programming environment (like WebStorm).
