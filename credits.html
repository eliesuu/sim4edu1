Credits

The OESjs simulator uses a number of open source libraries:

  • cLASSjs, for model-based development.
  • SIM.JS Random library, based on a JS port of the C implementation of the Mersenne Twister algorithm by Makoto Matsumoto and Takuji Nishimura.
  • PrismJS, for JavaScript code highlighting.
  • Chartist.js, a small footprint charting library based on SVG.
  • Phaser.js, a 2D game engine used for 2D visualisation with the HTML5 canvas 2D context.
  • Babylon.js, a 3D game engine used for 3D visualisation with the HTML5 canvas 3D context based on WebGL.
